Available examples¶

It is not allowed to use any of the examples materials’ outside of the scope of this tutorial. Experimental data and models distributed in the examples are even wrong or tricked to serve the purpose of learning fullrmc.

Herein is a list of all available examples in the standard distribution. A short description of each example is shown in the list below. It is recommended not to run any example from within fullrmc. But copy the example folder to any safe place such your home directory and run it from there. To locate an example and copy it, one can do the following:

# import find_example
from fullrmc.Examples import find_example
# search for an examples. e.g. atomicNiTi
print find_example('atomicNiTi')

>>> '../site-packages/fullrmc/Examples/atomicNiTi'

In all examples, run.py is the name of the file that muse be executed from the command line as the following:

python run.py

Examples folder name is used as the click-able items in the following list.

  1. translations: Used show how to use translation generators.
  2. rotations: Used to show how to use rotation generators.
  3. agitations: Used to show how to use agitation generators.
  4. docking: To show how to move one group of atoms towards another.
  5. explore: This is a simulation of a molecule exploring a cavity.
  6. refine: This is a simulation of a molecule refining its position in a cavity.
  7. benchmark: This example can be used to benchmark fullrmc.
  8. MLSelection: In this example we show that reinforcement machine learning can help increasing accepted moves ratio.
  9. ExploreCO2: This is a fit of CO2 gas in critical condition.
  10. atomicNiTi: Fitting a Ni-Ti memory shape binary compound.
  11. molecularTHF: This is a tetrahydrofuran electrolyte system fitting.
  12. SiOxNanosphere: This is an isolated silicon oxide nanosphere fitting.
